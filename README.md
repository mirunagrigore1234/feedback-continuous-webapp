# Continuous Feedback WebApp

## Descriere
Aceasta este o aplicație web pentru acordarea de feedback continuu la cursuri și seminarii. Aplicația permite studenților să reacționeze la activități prin emoticoane, iar profesorii pot vizualiza feedback-ul anonim în timp real sau ulterior.

## Tema
Continuous Feedback Application (Tema 6 proiect Tehnologii Web ASE)

## Funcționalități minime
- Profesorul definește o activitate cu o dată și un cod unic de acces.
- Studentul introduce codul pentru a participa la activitate.
- Studentul poate trimite feedback prin alegerea unor emoticoane (smiley, frowny, surprised, confused).
- Profesorul poate vizualiza feedback-ul anonim asociat cu momentul de timp.
- Feedback-ul este disponibil în timpul activității și ulterior.
